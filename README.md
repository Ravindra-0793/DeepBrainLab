# The Switchboard Test – Data and Code Repository

This repository contains metadata files and experimental control code related to *The Switchboard Test*, a flexible, low-cost setup for assessing spatial memory in rats during sequential instrumental tasks.

##  Data Availability

Representative sample datasets and task execution code are publicly available at Figshare:

https://doi.org/10.6084/m9.figshare.31034230

The Figshare record is the authoritative source for publicly released data associated with this project. Full experimental datasets may be made publicly available in future releases and are currently available from the authors upon reasonable request.

## Data Pipeline and Metadata

This repository includes JSON metadata descriptions for different stages of the data pipeline:
- Raw behavioral event logs
- Analyzed behavioral performance data
- Processed DeepLabCut tracking data
- Merged behavioral and DeepLabCut tracking data (±100 ms temporal alignment)
- Arduino-based task execution and event-logging code
